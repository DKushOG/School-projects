## Featured Projects

### Event Manager Web Application
**Tech stack:** Node.js, Express.js, EJS, SQLite

A full-stack web application that supports organiser and attendee workflows for creating, publishing, and booking events.  
Key features include:
- organiser and attendee views
- event creation and management
- bookings and site settings
- session-based organiser authentication
- protected routes and relational database design

### Otodecks DJ Application
**Tech stack:** C++, JUCE

A desktop DJ application developed using object-oriented programming principles.  
Key features include:
- multi-deck playback
- waveform visualisation
- drag-and-drop audio loading
- persistent audio library
- looping controls
- optional third deck support

### Theme Park Website
**Tech stack:** HTML, CSS, JavaScript

A multi-page website developed as a team project.  
My main contribution was the **Food & Drink** section, which included:
- carousel banner content
- interactive image cards
- internal page navigation
- restaurant map layout
- semantic HTML and reusable CSS styling
