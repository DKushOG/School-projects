# Otodecks DJ Application

A desktop DJ application built with **C++ and JUCE**, developed using object-oriented programming principles.

## Features
- multi-deck playback
- waveform visualisation
- drag-and-drop loading
- persistent audio library
- looping controls
- optional third deck

## Tech Stack
- C++
- JUCE

## Notes
This project focused on GUI programming, modular software design, and event-driven interaction.
