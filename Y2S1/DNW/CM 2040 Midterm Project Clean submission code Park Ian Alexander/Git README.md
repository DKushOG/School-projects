# Event Manager Web Application

A full-stack event management web application built with **Node.js, Express.js, EJS, and SQLite**.

## Features
- organiser and attendee workflows
- event creation, publishing, and editing
- ticket bookings
- site settings
- session-based organiser authentication
- protected routes
- relational database design

## Tech Stack
- Node.js
- Express.js
- EJS
- SQLite

## Notes
This project was developed as coursework and focused on server-side rendering, database integration, and user workflow design.
